Projeto pula pula

Controlar a entrada e saída das crianças

Requisitos
-> data de acompanhamento (diaria)
-> de uma criança
    -> nome ou do responsável
-> tempo (tempo mínimo recorrente 10 minutos) 
    ---> calcular o tempo a partir da hora de entrada - o total de minutos
    ---> ações de encerrar ou prolongar minutos
    resultado 
        -> alerta
        -> recurso visual para saber se a criança esta brincando
    -> iniciar contagem
    -> parada (automatica)
    -> pode ser prorrogado +10


    Valores
    -> valor (calculo)
    -> forma de pagamento
        -> pix / especie / outros

Fluxo de utilização:
- Ter uma tela inicial onde o app possa ser inicializado
    - Inserir o nome do operador

- Essa inicialização deve salvar a data atual, indicando que o app esta em uso



Entidades: 
* Criança:
    - nome
    - hora de entrada
    - hora de saída
    - tempo (minutos)
